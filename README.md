## Development

Install dependencies:
```shellscript
npm install
```

Run the dev server:

```shellscript
npm run dev
```

Deployment

Build for production:
```shellscript
npm run build
```

Start in production mode:
```shellscript
npm start
```
Deploy the contents of build/server and build/client to your host of choice.
